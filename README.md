This is a simple Weather Application built using Python and Tkinter for the GUI.
It fetches real-time weather data using the OpenWeatherMap API and displays the temperature and weather description of a user-specified city.

* Features

User-friendly graphical interface using Tkinter

Fetches live weather information by city name

Displays temperature (in Celsius) and weather condition

Handles invalid input and network errors gracefully

* Tech Stack

Python 3

Tkinter (for GUI)

Requests (for API calls)

OpenWeatherMap API

⚙️ How to Run

1) Clone this repository

2) Install required dependencies:

pip install requests


3) Run the script:

python weather_app.py


4) Enter a city name and click "Get Details" to view weather info.
